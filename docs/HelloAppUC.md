# HelloApp Use Cases

## UC1: Display "Hello World"
Print a basic greeting in the console.

## UC2: Display "Hello" with Command-Line Argument
Accept one name via command-line input and greet that user.

## UC3: Display "Hello" with Command-Line Argument or Default Message
Support optional argument handling with a default greeting path.

## UC4: Display "Hello" with Multiple Command-Line Arguments
Handle multiple command-line names in one execution.

## UC5: Read Name from Standard Input
Read a single name from standard input.

## UC6: Read Multiple Names from Standard Input
Read and process multiple names from standard input.

## UC7: Store Names in Memory
Store entered names in memory and list them on request.

## UC8: Remove Names
Add removal support for stored names.

## UC9: Extract Methods
Extract input-processing logic into dedicated methods.

## UC10: Separate Class
Move name-management responsibilities into a separate class.

## UC11: Persistence
Persist names to storage and reload them across runs.

## UC12: Banner Display
Render greeting text in banner-style output for enhanced display.
